# Day 2: Coffee Barista Voice Agent

## Objective
Build a coffee ordering voice agent that takes complete orders and saves them to JSON

## Core Features
- [x] Coffee drink selection (latte, cappuccino, americano, etc.)
- [x] Size selection (small, medium, large, extra large)
- [x] Milk type selection (whole, skim, almond, oat, soy, coconut, none)
- [x] Extras selection (sugar, whipped cream, caramel, extra shot, etc.)
- [x] Customer name collection
- [x] Order completion and JSON persistence
- [x] HTML visualization generation

## Implementation
- Function tools for each order component
- JSON file storage for order history
- HTML visualization with beverage graphics
- Friendly barista personality and conversation flow

## Files
- Backend: Modified `src/agent.py` with barista agent
- Data: `orders.json` for order persistence
- Visualization: Auto-generated HTML files

## Testing
1. Start agent and have coffee ordering conversation
2. Complete full order with all details
3. Verify JSON file creation and HTML visualization
4. Test order status and visualization functions
